## üéØ Quick Start

```bash
# Clone the repository
git clone https://github.com/KRI5HNA-04/Kursor.git

# Navigate to the main application
cd Kursor/kursor-editor

# Install dependencies
npm install

# Set up environment variables (see kursor-editor/README.md)
```
